## ⚠️ 下载注意事项

1. **arXiv 论文可直接下载 PDF**：在链接后加 `/pdf`，如 `https://arxiv.org/pdf/2506.00073`
2. **付费墙（paywall）问题**：Springer、Elsevier、SAGE、ACM 等期刊/会议论文可能需要机构订阅。可通过以下方式获取：
   - 在 arXiv 搜索同篇论文的预印本版本
   - 通过学校图书馆的数据库访问
   - 使用 Google Scholar 查找作者个人主页上的公开版本
3. **中国期刊**：中国科学基金、系统管理学报等中文期刊可通过 CNKI（知网）下载
4. **GitHub 项目**：直接 clone 或下载 ZIP
